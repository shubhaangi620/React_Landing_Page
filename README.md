# üåê React + Tailwind CSS Landing Page

A modern, responsive, and visually appealing **landing page** built using **React** and **Tailwind CSS**.  
This project focuses on clean UI, smooth layout, and a mobile-first design approach.



##  Features

-  Built with **React**
-  Styled using **Tailwind CSS**
-  Fully responsive (mobile, tablet & desktop)
